## Overview

Hotjar MCP server provides AI-powered access to behavior analytics and user feedback tools, enabling website optimization through heatmaps, recordings, and surveys.

## Features

- **Heatmaps**: Visualize where users click, move, and scroll
- **Session Recordings**: Watch how users navigate your site
- **Feedback Widgets**: Collect user opinions and ratings
- **Surveys**: Ask users questions at key moments
- **Funnel Analysis**: Identify drop-off points in flows

## Platform Capabilities

Hotjar helps teams understand user behavior and gather feedback to improve website usability and conversion rates.

## Use Cases

- Optimizing landing page conversions
- Understanding navigation patterns
- Identifying usability issues
- Gathering user feedback
- A/B test validation

## Key Benefits

- Easy setup and installation
- Affordable pricing for SMBs
- Visual behavior insights
- Direct user feedback
- GDPR compliant

## MCP Integration

Enables AI assistants to query heatmap data, analyze session recordings, review feedback, and generate UX insights through natural language.

## Target Audience

Popular with marketers, UX designers, and product managers looking for accessible behavior analytics without extensive technical setup.

## Comparison

More accessible and affordable than enterprise solutions like FullStory, focused on essential behavior analytics and feedback collection.
