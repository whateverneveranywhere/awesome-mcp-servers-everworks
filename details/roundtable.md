# Roundtable

Multi-model AI debate platform with a remote MCP server. GPT-4o, Claude, Gemini and 200+ models discuss your prompt in structured rounds, then a moderator synthesizes collective insight.

## MCP Endpoint

- **URL**: `https://mcp.roundtable.now/mcp`
- **Transport**: Streamable HTTP
- **Auth**: API Key via `X-API-Key` header

## Tools

| Tool | Description |
|------|-------------|
| `consult_council` | Get multiple AI perspectives on any question through structured multi-model debate |
| `design_architecture` | Architectural recommendations from multiple AI models for system design decisions |
| `review_code` | Multi-perspective code review from multiple AI models |
| `debug_issue` | Debug issues using multiple AI models from different angles |
| `plan_implementation` | Implementation planning using multi-model consensus |
| `assess_tradeoffs` | Evaluate technical tradeoffs using multiple AI perspectives |
