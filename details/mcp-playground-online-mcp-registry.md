## Overview

MCP Playground Online hosts the most comprehensive MCP server directory, featuring 10,000+ MCP servers. It allows users to find integrations for Claude, Cursor, and any MCP-compatible client. Each server provides tools, prompts, or resources that AI models can use.

## Features

- Over 10,000 MCP servers listed
- Categorized browsing: Database, Developer Tools, Productivity, Cloud Services, File Systems, Search & Data, and more
- Free MCP testing tool to test servers instantly
- Support for Claude Desktop, Cursor IDE, and custom apps via MCP SDK
- Publishing guide for contributors to share their own MCP servers

## Use Cases

- Discover MCP servers for extending AI assistant capabilities
- Test MCP servers before integration
- Find specific integrations like Supabase, PostgreSQL, MongoDB, GitHub, GitLab, Playwright, Notion, Slack, Linear, Figma, AWS, Google Cloud, Cloudflare, Local FS, Google Drive, Dropbox, Brave Search, Exa, Scraping, etc.

## Pricing

Free to use. No cost for browsing, searching, or testing MCP servers.

## How to Use

- Claude Desktop: configure via `claude_desktop_config.json`
- Cursor IDE: set up in `.cursor/mcp.json`
- Custom Apps: integrate using the MCP SDK

## Contributing

Built an MCP server? Publish it to share with the community. Learn more in the publishing guide available on the site.
