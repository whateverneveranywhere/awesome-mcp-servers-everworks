## Features

- **firecrawl_scrape**: Scrape any URL into clean markdown or structured JSON with options for mobile rendering, tag filtering, JavaScript waiting.
- **firecrawl_search**: Web search with content extraction and time-based filtering.
- **firecrawl_crawl / firecrawl_check_crawl_status**: Asynchronous site crawling with depth, deduplication, domain constraints.
- **firecrawl_map**: Discover indexed URLs on a site.
- **firecrawl_extract**: LLM-based structured data extraction using JSON schema.
- **firecrawl_agent / firecrawl_agent_status**: Autonomous research agent for browsing, searching, compiling reports.
- **firecrawl_interact**: Interactive page actions like clicking, form filling using natural language or Playwright; supports logged-in sessions.
- **firecrawl_browser_* tools**: Persistent browser session management via CDP.
