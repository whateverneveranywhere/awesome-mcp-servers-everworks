# Godot

MCP server specialized for Godot Engine projects, providing knowledge of file formats (.gd, .tscn, .tres), architecture patterns, pitfalls, templates, and CLI workflows. Includes the `godot` command for running, validating, importing, and exporting. Ideal for game development, debugging, and implementing systems.
