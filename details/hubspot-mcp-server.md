# HubSpot MCP Server

HubSpot MCP Server provides read-only access to CRM data, enabling AI to analyze deals, tickets, trends, associations, and pipeline snapshots without exposing sensitive information. It comes in remote and local developer versions, requiring a HubSpot Developer account. Ideal for teams relying on HubSpot for CRM insights and reporting.

[https://developers.hubspot.com/docs/api/crm](https://developers.hubspot.com/docs/api/crm)
