## Overview

Heap MCP server provides AI-powered access to auto-capture analytics capabilities, enabling analysis of complete user behavior without manual event instrumentation.

## Features

- **Auto-Capture**: Track every user interaction automatically
- **Retroactive Analytics**: Analyze historical data without prior setup
- **Session Replay**: Watch recordings of user sessions
- **Path Analysis**: Understand user journey flows
- **Data Science Tools**: SQL access and data export

## Platform Capabilities

Heap automatically captures all user interactions, eliminating the need for manual event tracking and enabling retroactive analysis of historical behavior.

## Use Cases

- Analyzing user behavior without instrumentation
- Retroactive event analysis
- Understanding conversion paths
- Debugging user experience issues
- Product optimization research

## Key Benefits

- No manual event tracking required
- Complete data capture from day one
- Retroactive data analysis
- Session replay for context
- Data science-friendly exports

## MCP Integration

Enables AI assistants to query auto-captured data, analyze user paths, access session replays, and extract behavioral insights through natural language.

## Comparison

Differentiated from Mixpanel and Amplitude by auto-capture approach, eliminating instrumentation overhead and enabling retroactive analysis of any user action.
