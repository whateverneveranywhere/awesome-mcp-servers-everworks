## Overview

Figma's official Dev Mode MCP server provides full structural details of selected layers in Figma.

## Features

- Expose selected Figma layers with hierarchy and properties
- Surface design tokens, colors, typography
- Component variants and properties
- Auto-layout constraints and spacing rules

## Use Cases

- Generate React components from Figma designs using Tailwind
- Extract spacing and color tokens
- Create CSS matching Figma typography

## Pricing

Included with Figma Dev Mode access.
