## Overview

mcpservers.org is a curated directory of MCP (Model Context Protocol) servers enabling AI assistants to interact with various services, tools, and data sources.

## Key Items

- **Bright Data** (sponsor): Web discovery, extraction, interaction.
- **Scout Monitoring MCP** (sponsor): Performance and error data for AI assistants.
- **Alpha Vantage MCP Server** (sponsor): Financial market data including stocks, ETFs, options, forex, crypto.
- **Anki MCP** (official): Interaction with Anki spaced repetition flashcards.
- **Browserbase** (official): Cloud-based browser automation for navigation, extraction, form filling.
- **Chrome DevTools MCP** (official): Control and inspect live Chrome browser for coding agents.
- **Cloudflare** (official): Deploy and manage Workers, KV, R2, D1 resources.
- **Context 7** (official): Up-to-date docs for Cursor prompts.
- **DeepWiki by Devin** (official): AI-powered codebase context.
- **E2B** (official): Secure code sandboxes.
- **Exa** (official): AI search engine.
- **Firecrawl MCP** (official): Web scraping and search for LLMs.
- **GitHub** (official): GitHub MCP Server.
- **Google MCP Servers** (official): Google's MCP servers collection.
- **MiniMax MCP** (official): Text-to-speech, image, video generation.
- **Next.js DevTools MCP** (official): Next.js tools for AI coding assistants.
- **NotebookLM MCP Server**: Chat with NotebookLM notebooks.
- **Playwright** (official): Playwright browser automation.
- **Proxyman MCP** (official): Inspect HTTP traffic, debugging rules.
- **Supabase** (official): Database, auth, edge functions.
- **XcodeBuildMCP**: Tools for iOS/macOS projects.
- **ContextPulse**: Local ambient context (screen, OCR, voice, tracking).
- **systemprompt**: Self-hosted MCP governance in Rust.
- **Pinterest Ads MCP**: Ads insights via Two Minute Reports.
- **Twitter (X) Ads MCP**: Ads tracking via Two Minute Reports.
- **ChilledSites MCP Server**: AI website builder.
- **Perspective AI**: AI concierge for forms.
- **HuiMei Social Media Automation MCP Server**: Publish to 12+ platforms.
- **InfoLeg**: Wrapper for infoleg.gob.ar legal site.

## Pricing

Mostly free/open-source; some sponsored or with usage-based pricing.
