## Overview

PocketBase MCP server provides AI-powered access to a single-file backend solution with embedded database, authentication, file storage, and admin UI.

## Features

- **Single Executable**: Everything in one Go binary
- **Embedded Database**: SQLite-based data storage
- **Authentication**: Built-in user auth and OAuth2
- **File Storage**: Local and S3-compatible storage
- **Admin UI**: Web-based admin dashboard
- **Real-time**: WebSocket subscriptions

## Platform Capabilities

PocketBase is an open-source backend in a single file, perfect for prototypes, MVPs, and small to medium applications.

## Use Cases

- Rapid prototyping
- Small to medium web applications
- Mobile app backends
- Side projects and MVPs
- Self-hosted personal apps

## Key Benefits

- Extremely simple deployment (one file)
- No external dependencies
- Built-in admin dashboard
- Easy to extend with Go
- Very low resource usage

## MCP Integration

Enables AI assistants to manage collections, configure authentication, handle files, and query data through conversational interfaces.

## Philosophy

Designed to be the simplest backend solution possible while still providing essential features for modern applications.

## Deployment

Can be deployed anywhere that runs Go binaries, including serverless platforms, VPS, or even embedded devices.
