# mcp-openapi

Turn any OpenAPI spec into callable MCP tools for Claude.

## Features

- Converts any OpenAPI 3.x specification into MCP tools
- Enables Claude to call every endpoint through natural language
- Supports Bearer token authentication
- Supports API key authentication
- Supports Basic authentication
- Point at any OpenAPI spec URL or file

## Usage

Point the server at any OpenAPI 3.x specification to automatically generate MCP tools for all endpoints.

## Authentication

Supports multiple authentication methods:
- Bearer tokens
- API keys
- Basic auth

## Compatibility

Works with any OpenAPI 3.x compliant specification.
