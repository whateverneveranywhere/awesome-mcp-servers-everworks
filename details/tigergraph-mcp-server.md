## Overview

TigerGraph MCP server provides AI-powered access to enterprise graph analytics platform with real-time deep link analytics and parallel graph processing.

## Features

- **Deep Link Analytics**: Analyze connections across many hops
- **Parallel Processing**: Native parallel graph computation
- **Real-Time Updates**: Sub-second graph updates and queries
- **GSQL**: Purpose-built graph query language
- **Machine Learning**: Built-in graph ML capabilities

## Platform Capabilities

TigerGraph is an enterprise graph database and analytics platform designed for large-scale graph workloads requiring deep link analysis.
