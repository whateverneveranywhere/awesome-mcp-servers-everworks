# Ahrefs MCP Server

## Overview
The Ahrefs MCP Server connects AI models to Ahrefs' SEO data via the Model Context Protocol. It allows users to perform keyword research, analyze competitor rankings, and get live traffic metrics using natural language, leveraging Ahrefs API v3.

## Features
- Live access to Ahrefs SEO metrics (keyword rankings, backlink data, organic traffic)
- Natural language querying for SEO insights (e.g., "Show me keyword rankings for competitor X")
- Supports Ahrefs API v3 for reliable data access
- Enables competitor gap analysis and content strategy planning
- Read-only access to protect data integrity

## Use Cases
- SEO specialists conducting keyword research via AI chat
- Content strategists identifying ranking opportunities
- Marketing teams monitoring competitor performance

## Pricing
Access to the Ahrefs MCP Server requires an Ahrefs Enterprise Plan. Pricing is based on Ahrefs' subscription tiers; the MCP server functionality is included with the Enterprise Plan.
