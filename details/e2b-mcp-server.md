## Overview

E2B provides secure cloud sandboxes for AI agents to run code safely.

## Features

- Execute Python and JavaScript in isolated cloud sandboxes
- Run shell commands with stdout/stderr inspection
- Install packages and manage dependencies
- Persist sandboxes across tool calls for sessions

## Use Cases

- Data processing and analysis scripts
- Testing regex or logic against edge cases
- Running database migrations on snapshots

## Pricing

Usage-based pricing with free tier available.
