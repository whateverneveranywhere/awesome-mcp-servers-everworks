# UI/UX Pro Max

MCP server providing UI/UX design intelligence with 50 styles, 21 palettes, 50 font pairings, 20 charts, and support for stacks like React, Next.js, Vue, Svelte. Handles actions such as plan, build, design, review UI/UX code for websites, apps, elements like buttons, modals, and styles including glassmorphism, neumorphism.
